# cam-platform-risk-score
Este repositório faz parte do projeto **CamFaro** e tem como objetivo criar um sistema de avaliação de risco para plataformas de webcam.

## 📌 Problema

Usuários frequentemente entram em plataformas sem saber:
- se são confiáveis  
- se existem práticas abusivas  
- se há riscos de gasto excessivo  

## 🎯 Objetivo

Desenvolver um modelo simples para classificar o nível de risco de uma plataforma.

## ⚙️ Critérios de Avaliação

### 1. Transparência
- Preços claros
- Informações acessíveis

### 2. Comportamento da Plataforma
- Upsells agressivos
- Estratégias de retenção

### 3. Perfis e Conteúdo
- Perfis suspeitos
- Atividade automatizada

### 4. Feedback de Usuários
- Reclamações
- Experiências negativas

### 5. Sistema de Pagamento
- Segurança
- Clareza nas cobranças

## 📊 Score

Cada plataforma recebe uma pontuação baseada nesses critérios, permitindo:

- comparação rápida
- identificação de riscos
- decisões mais informadas

## 🧠 Filosofia

O objetivo não é dizer o que usar, mas ajudar o usuário a entender o risco envolvido.

## 🌍 Contexto

O mercado de plataformas ao vivo ainda é pouco transparente, principalmente para usuários fora dos principais mercados.

## 🚀 Evolução

- Score automático
- Integração com análises do CamFaro
- Base de dados de plataformas
